##Facial Emotion Detection Project

Description
โปรเจคนี้เป็นระบบตรวจจับอารมณ์จากใบหน้า (Facial Emotion Detection) โดยใช้เทคนิคด้าน Computer Vision และ Deep Learning
สามารถวิเคราะห์อารมณ์ของบุคคลจากภาพหรือวิดีโอ เช่น 😊 Happy 

Features
ตรวจจับใบหน้าจากภาพและวิดีโอ
จำแนกอารมณ์แบบ Real-time
รองรับการใช้วิดีโอหรือกล้อง
สามารถนำไปต่อยอดกับ Dashboard (เช่น Node-RED)

Technologies Used
Python
OpenCV
YOLO (You Only Look Once)
Deep Learning Model
